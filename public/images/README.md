# Images Directory

This directory contains all images used throughout the yourdivorcerealtor.com website.

## Folder Structure

- **hero/**: Homepage and page hero images
- **agent/**: Dr. Jan Duffy professional photos
- **communities/**: Las Vegas community/city photos (12 cities)
- **process/**: Divorce process step-by-step images
- **services/**: Service offering images
- **icons/**: Icon graphics and illustrations
- **testimonials/**: Client testimonial photos (with permission)
- **social/**: Social media sharing images (OG, Twitter cards)
- **logo/**: Logo files (SVG, PNG)
- **resources/**: Resource page visuals (calculators, checklists, timelines)
- **staging/**: Home staging before/after photos

## Image Guidelines

- **Format**: WebP preferred, JPG fallback
- **Size**: Optimized for web (< 200KB per image)
- **Naming**: Use kebab-case (e.g., `dr-jan-duffy-headshot.jpg`)
- **Alt Text**: Always include descriptive alt text in components
